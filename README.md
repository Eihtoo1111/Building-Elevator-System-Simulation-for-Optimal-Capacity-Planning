# Building-Elevator-System-Simulation-for-Optimal-Capacity-Planning
This is a realistic simulation to determine how many elevators a new five‑floor commercial building should have, balancing the high installation cost with the need to minimize user waiting times.
Since elevator behavior depends on complex factors such as
variable traffic patterns, elevator capacity, floor‑to‑floor travel times, and uneven demand across floors,
this simulation will accurately model elevator operations and incorporate the 15‑minute traffic data xslx file.
